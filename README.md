# Setting Up and Managing the Repository for Local Development

## Initial Setup

*Prerequisites:* Must have node installed (https://nodejs.org/en/) - Works with node version ```v14.17.0```

1. Clone repository to your local computer
2. Find the root folder where "gulpfile.js" file is available
3. Run ``` npm install ```
4. The above command should install node modules folder within your project.
5. After the successful installation of node modules, run ``` gulp ``` to watch the changes
